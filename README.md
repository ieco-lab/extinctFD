# leiocephalus_nonrandom_extinction
Submitted data and code for *Detecting functional diversity loss under directional, stabilizing, and disruptive models of anthropogenic extinction* 

Data have not been reviewed and have not been published. Until that time the methods to produce the data and its citation is below

## Citation

Huron, NA, Hedges, SB, Helmus, MR. 2022. Detecting stabilizing, directional, and disruptive
patterns of anthropogenic species loss with general models of nonrandom extinction. bioRxiv.
[doi:10.1101/2022.09.11.507476](https://doi.org/10.1101/2022.09.11.507476).
